Design notes — the right panel re-renders the exact same React components the public-facing /school/[id] page uses, so there is no divergence. Sectioned white cards on a cream backdrop keeps the editor warm (vs. the cold gray surface admin pattern most SaaS reaches for by default). Character counters use the same 0.55rem uppercase tracking-widest micro-label style as the rest of the site.

Design language recap

Reusing the same kit — not a separate app.

palette

Same palette

Cream #fdf9f0 surface, crimson #b40f55 primary, amber #7d570e secondary. No "admin gray" anywhere. The dashboard is as warm as the marketing site.

text_fields

Same type

Plus Jakarta Sans extrabold + italic emphasis on every main heading (“Good morning, Ava”). Be Vietnam Pro for body. Uppercase micro-labels at 0.6rem / tracking-widest are the signature rhythm.

rounded_corner

Same geometry

Generous radii 1–2rem on cards, full pills on chips and buttons. Every card wears editorial-shadow + border-[#FAEEDA]. No square SaaS tables.

extension

Same components

The ClassCard in the editor preview is the same React component students see on /school/[id]. One source of truth — less drift, easier maintenance.
